Descrição

Você quer se tornar um Programador WEB?

Qualquer que seja sua motivação, bem-vindo ao nosso curso de programação web.

Este curso de Desenvolvedor Web vai proporcionar tudo o que você precisa saber para criar seu próprio website.

Neste curso cobriremos em mais de 12 horas de conteúdo os seguintes tópicos:


1) HTML5, CSS3 e JavaScript com os comandos mais importantes.
2) Criar um site completo e responsivo que se adapta no Tablet e Smartphone e em telas maiores.
3) Trabalhar com imagens, áudio, vídeo usando boas práticas.
4) Aprender a criar um carrinho de compras completo com cálculo de frete para uma loja virtual, utilizando PHP e MySQL.
